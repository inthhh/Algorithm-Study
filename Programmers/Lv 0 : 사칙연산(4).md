두 수의 합
```
int solution(int num1, int num2) {
    int answer = -1;
    answer = num1 + num2;
    return answer;
}
```

두 수의 차
```
int solution(int num1, int num2) {
    int answer = 0;
    answer = num1 - num2;
    return answer;
}
```

두 수의 곱
```
int solution(int num1, int num2) {
    int answer = 0;
    answer = num1 * num2;
    return answer;
}
```
